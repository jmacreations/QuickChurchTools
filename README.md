A collection of simple, lightweight, and privacy-focused web tools designed to assist with common administrative tasks for churches and ministries.

### Tools

*   **[Shift m2M Categorisation Tool](https://jmacreations.github.io/QuickChurchTools/Shiftm2M/categorisation-tool.html)**: A simple, offline tool to help categorise people or programs into the 5 key areas of the Shift m2M framework. Import a CSV, categorise your data, and export the results. All data stays on your local machine.
